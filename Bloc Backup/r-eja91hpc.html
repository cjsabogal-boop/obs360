Manual Básico de Integración

Esta guía te ayudará a reemplazar tu "Sistema de Autenticación Simulado" por los servicios reales de Google Firebase. Aquí aprenderás a conectar, autenticar usuarios y guardar datos.

Nota: Este manual utiliza la versión Modular (v9+) del SDK de Firebase, que es la más moderna y eficiente.

Paso 1: Crear Proyecto y Obtener Credenciales

Antes de tocar código, necesitas configurar la nube:

  1. Ve a la Consola de Firebase.
  2. Haz clic en "Agregar proyecto" y sigue los pasos.
  3. En el menú lateral, ve a "Compilación" > "Authentication" y activa "Correo/Contraseña".
  4. Vuelve a "Visión general", haz clic en el icono Web (</>).
  5. Registra la app y copia el objeto firebaseConfig.

Paso 2: Instalación

Abre la terminal en la carpeta de tu proyecto y ejecuta:

npm install firebase

Si usas HTML puro sin Node.js, puedes usar el CDN (script tag), pero se recomienda npm para apps modernas.

Paso 3: Archivo de Configuración

Crea un archivo llamado src/firebase/config.js (o similar) y pega lo siguiente:

// Importar las funciones necesarias del SDK
import { initializeApp } from "firebase/app";
import { getAuth } from "firebase/auth";
import { getFirestore } from "firebase/firestore";

// Tu configuración web (Reemplaza con TUS datos de la consola)
const firebaseConfig = {
  apiKey: "AIzaSyD-Ejemplo-De-Tu-Key",
  authDomain: "tu-proyecto.firebaseapp.com",
  projectId: "tu-proyecto",
  storageBucket: "tu-proyecto.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456:web:abcdef"
};

// 1. Inicializar Firebase
const app = initializeApp(firebaseConfig);

// 2. Inicializar servicios y exportarlos
export const auth = getAuth(app);
export const db = getFirestore(app);
Seguridad: No subas este archivo a GitHub público. Usa variables de entorno (process.env) para las claves en un proyecto real.

Paso 4: Autenticación (Login)

Cómo reemplazar tu función simulada de Login por la real:

Función de Inicio de Sesión

import { signInWithEmailAndPassword } from "firebase/auth";
import { auth } from "./firebase/config";

const loginUser = async (email, password) => {
    try {
        const userCredential = await signInWithEmailAndPassword(auth, email, password);
        const user = userCredential.user;
        console.log("Bienvenido:", user.email);
        return user;
    } catch (error) {
        console.error("Error code:", error.code);
        console.error("Mensaje:", error.message);
        alert("Error al iniciar sesión: " + error.message);
    }
}

Función de Registro (Crear cuenta)

import { createUserWithEmailAndPassword } from "firebase/auth";
import { auth } from "./firebase/config";

const registerUser = async (email, password) => {
    try {
        const userCredential = await createUserWithEmailAndPassword(auth, email, password);
        console.log("Cuenta creada para:", userCredential.user.email);
    } catch (error) {
        console.error(error);
    }
}

Paso 5: Base de Datos (Firestore)

Ejemplo básico para guardar información de un usuario:

import { collection, addDoc, getDocs } from "firebase/firestore";
import { db } from "./firebase/config";

// Guardar datos
const guardarTarea = async (titulo) => {
    try {
        const docRef = await addDoc(collection(db, "tareas"), {
            titulo: titulo,
            completada: false,
            fecha: new Date()
        });
        console.log("Documento escrito con ID: ", docRef.id);
    } catch (e) {
        console.error("Error añadiendo documento: ", e);
    }
}

6. Usos y Aplicaciones Comunes

Firebase ofrece un ecosistema completo. Aquí tienes ejemplos de qué herramienta usar según tu necesidad:

Chat en Tiempo Real

Herramienta: Firestore o Realtime DB

Permite que los mensajes aparezcan instantáneamente en los dispositivos de otros usuarios sin recargar.

Subida de Fotos

Herramienta: Cloud Storage

Ideal para apps tipo Instagram donde los usuarios suben sus fotos de perfil o archivos.

Notificaciones

Herramienta: Cloud Messaging (FCM)

Para avisar al usuario de eventos importantes ("Tu pedido ha llegado") incluso con la app cerrada.

Lógica Automática

Herramienta: Cloud Functions

Ejecutar código backend (como enviar un email de bienvenida) automáticamente cuando alguien se registra.
