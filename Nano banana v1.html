No Designer. No Photographer.
Just You & The Model.

A definitive playbook for using Google's "Thinking with 3 Pro" (Codename: Nano Banana) to generate high-converting Amazon listing images from scratch.

The Setup

  • Download Google Gemini App (iOS or Android).
  • Switch model to "Thinking with 3 Pro".
  • Confirm you see the "Thinking" icon, not "Fast".

The Input Photos

  • Take 3-5 raw photos with your phone.
  • Include context clues (hands, table) for scale.
  • Ensure label text is legible if visible.

The Core Workflow

1

Upload & Initial Prompt

Upload your raw photos. Use natural language. Don't over-engineer the prompt. Start simple.

2

Iterate (The "New Chat" Rule)

If the result is good but needs tweaks, ask in the same chat.
CRITICAL: If the chat "goes off the rails" (results get worse), Stop. Download the last good image, start a New Chat, upload that image, and continue.

3

The "Download" Reality Check

The image in the chat preview is a low-res draft. Always download the image to see the final quality. The model applies "finishing" processing (sharpening text, fixing details) only on the downloaded file.

Troubleshooting

The "AI Gibberish" Fix

If the label text on your product comes out as alien language:

  1. Download the gibberish image.
  2. Start a NEW CHAT.
  3. Upload the gibberish image + a close-up photo of the real label text.
  4. Prompt: "Fix the text on the label to match the reference photo exactly."
  5. Download result to verify (chat preview might still look wrong).

Going "Off the Rails"

Sometimes the model gets confused after too many edit requests in one thread. The image starts glowing, warping, or losing quality.

The Rule: Never try to save a doomed chat. Download the last "decent" version, abandon the chat, and start fresh.
